ADMINISTRACCIÓN DE SERVIDORES

¿Cómo descargar e instalar VirtualBox?

Características

  • Software que simula una computadora real.
  • Crea una especie de entorno aislado en una computadora.
  • Permite configurar hasta 32 CPUs.
  • Compatibilidad con todo tipo de hardware.
  • Funciones de clonado de máquinas.
  • Permite agrupar máquinas virtuales.

¿Qué es VirtualBox?

VirtualBox es una aplicación que sirve para hacer máquinas virtuales con instalaciones de sistemas operativos. Esto quiere decir que si tienes un ordenador con Windows, GNU/Linux o incluso macOS, puedes crear una máquina virtual con cualquier otro sistema operativo para utilizarlo dentro del que estés usando.


Objetivo

Proporcionar un entorno de ejecución independiente de la plataforma de hardware y del sistema operativo, que oculte los detalles de la plataforma subyacente y permita que un programa se ejecute siempre de la misma forma sobre cualquier plataforma.


Requerimientos


Memoria RAM de 2 MB para Windows 7; 4 GB para Windows 8 ó Windows 10. Aunque lo aconsejable empieza por el doble de lo indicado.

Espacio libre en disco de mínimo 90 MB para cada máquina virtual. Pero cualquier virtualización de un sistema operativo mínimamente moderno va a ocupar 1 GB.

Procesador Intel o AMD con soporte de virtualización. Para que la máquina vaya mínimamente fluida se recomienda procesadores de más de dos núcleos.

Una resolución de pantalla 800 x 600.

Versiones



VirtualBox 5.0

La versión 5.0 soporta los sistemas operativos más recientes invitado o alfitrión:

  • Mac OS X Yosemite
  • Windows 10
  • Oracle Linux
  • Oracle Solaris
  • Otros sistemas operativos Linux y sistemas legados
Las nuevas capacidades de Oracle VM VirtualBox 5.0 incluyen:

  • Soporte de virtualización para Guests en Windows y Linux
  • Soporte de dispositivos USB 3.0
  • Encriptación de imagen de disco

VirtualBox 6.0

Esta versión presenta importantes mejoras, además de un nuevo diseño de la interfaz de usuario. En esta nueva versión se esfuerza el soporte 3D.
La emulación de Hyper-V compatible con Windows como un núcleo de ejecución alternativo. Ayuda a solucionar problemas al ejecutar invitados de 64 bits cuando se habilitó la función de sandboxing de Seguridad de Windows.
