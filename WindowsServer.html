¿Qué es WINDOWS SERVER 2019?


Windows Server es una distribución de Microsoft para el uso de servidores. Está desarrollado en el lenguaje de programación C++ y Asembler. Se trata de un sistema multiproceso y multiusuario que a día de hoy utilizan millones de empresas de todo el mundo gracias a las características y ventajas que ofrece.


Características

  • Mejoras de calidad
  • Migraciones simples con servicio de almacenamiento de información
  • Subsistema de Windows para Linux
  • Sistema híbrido entre los entornos locales y los servidores de Azure
  • Plataforma de aplicaciones
  • Enhahced Storage Spaces Direct(S2D)
  • Windows Server 2019 Standard y Windows Server 2019 Datacenter

Requerimientos

  • Procesador de 64 bits a 1,4 GHz
  • Compatible con el conjunto de instrucciones x64
  • Admite DEP y NX
  • Admite CMPXCHG16b, LAHF/SAHF y PrefetchW
  • Admite la traducción de direcciones de segundo nivel (EPT o NPT)

Versiones

  • Windows Server 2019 Essentials
  • Windows Server 2019 Standard
  • Windows Server 2019 Datacenter
